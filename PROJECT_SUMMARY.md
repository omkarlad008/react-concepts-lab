# React Concepts Lab — Project Summary

## Overview

React Concepts Lab is a practical learning project built with React, TypeScript, Vite, React Router, Context API, localStorage, Vitest, and React Testing Library.

The purpose of this project is to practise core React concepts through a real, working task and learning dashboard instead of learning concepts only in isolation.

## Main Features

- Dashboard with task summary cards
- Task creation form using controlled components
- Task filtering by all, active, and completed status
- Dynamic task detail pages using URL params
- Multiple pages using React Router
- API practice page with loading, success, empty, and error states
- Global theme switching using Context API
- Persistent tasks and theme using localStorage
- Task state management using useReducer
- Performance optimisation using useMemo, useCallback, and React.memo
- Automated tests using Vitest and React Testing Library
- Basic accessibility improvements such as semantic structure and skip link
